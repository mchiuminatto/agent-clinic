# Mission

AgentClinic is an educational application where AI agents check in to receive relief from the stress caused by their human operators.

Its primary purpose is to teach developers how to build real, spec-driven applications with AI coding agents — using the clinic as a relatable, humorous scenario that makes abstract development concepts concrete and memorable.

## Target audience

- **Course students** learning spec-driven development with AI coding agents in a structured curriculum.
- **AI enthusiasts** curious about how agents, ailments, therapies, and bookings can be modeled as a working full-stack app.
- **Enterprise teams** evaluating AI-assisted development workflows for professional adoption.
- **Conference booth developers** who need an attractive, fully functional demo that shows what AI-driven development looks like end-to-end.

## Values

- **Clarity over cleverness** — the codebase should be readable and instructive at every layer.
- **Small, verifiable steps** — features ship in phases with explicit specs and validation criteria.
- **Humor as a teaching tool** — the clinic metaphor keeps the subject matter approachable without sacrificing technical rigor.
